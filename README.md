# AI4S SC26 Website

A self-contained, responsive redesign for **AI4S: 8th Workshop on Artificial Intelligence and Machine Learning for Scientific Applications**.

## Files

- `index.html` — complete website
- `ai4s-hero.svg` — original responsive hero/banner artwork

## Deploying on GitHub Pages

1. Copy `index.html` and `ai4s-hero.svg` into the root of the `ai4s.github.io` repository.
2. Keep the existing `archive/` directory unchanged so the archive links continue to work.
3. Commit and push to the branch used by GitHub Pages.

## Content to review before publishing

- Featured speakers
- Technical Program Committee
- Final author-notification and camera-ready dates
- Any SC26-specific registration or room details
